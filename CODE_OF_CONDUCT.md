# Code of Conduct

This project follows the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Summary

Behave decently. Assume the other person is operating in good faith until you have clear evidence otherwise. Disagreement on ideas is welcome; personal attacks are not.

## Scope

Applies to all project spaces: issues, pull requests, discussions, commit messages, and any related off-platform communication that relates to this project.

## Enforcement

Report violations by opening an issue or contacting the maintainers via the repo's GitHub contact channels. Reports are treated confidentially when possible. Maintainers may remove, edit, or reject contributions that don't follow this code, and may ban repeat violators.

For the full text of rights, responsibilities, and enforcement guidelines, see the link above.
